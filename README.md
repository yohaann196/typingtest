# Typing Test

A **dark-blue themed typing test** that tracks typing speed and accuracy. Features **live charts**, **CSV export**, and a readable interface.

## Demo

Try the live demo here:  
[https://yohaann196.github.io/typingtest/](https://yohaann196.github.io/typingtest/)

## Features

- Typing test with **100 words**  
- Tracks **WPM (Words Per Minute)** and **Accuracy (%)**  
- **Live charts** for speed and accuracy  
- **CSV export** of typing session results  
- Responsive layout with charts **side-by-side**  
- Dark-blue, minimal, readable design  

## Technologies Used

- **HTML**  
- **CSS**  
- **JavaScript**
